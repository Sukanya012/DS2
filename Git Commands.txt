Git SetUp
--------------
1) Install bash from  https://git-scm.com/downloads
2) launch bash
3) cd \documents
4) git init
5) ssh-keygen -t rsa -b 4096 -C "sukanya.anand12@gmail.com"
6) clip < ~/.ssh/id_rsa.pub
7) paste the keys in notepad

8) Login to github account
9) Goto Profile --> Settings --->  ssh and gpg keys
10) Add New Key -- Give a title and paste the Key and save
